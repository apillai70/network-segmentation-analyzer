**Shapes by Server Type:**

- **Inverted Trapezoid** = Load Balancers (F5, Traffic Manager)
- **Stadium (Pill)** = Monitoring (Splunk)
- **Flag** = Security (CyberArk, Azure Key Vault, Rapid7, DB Auditor)
